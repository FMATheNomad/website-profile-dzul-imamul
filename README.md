# Dzul Imamul Hakim — Master Barber

Website profil portofolio untuk **Dzul Imamul Hakim**, seorang master barber dan gentleman stylist. Halaman tunggal (single page) dengan tampilan elegan, maskulin, dan responsif.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS 3 + shadcn/ui
- **Icons:** Lucide React
- **Deploy:** Netlify (static export)

## Fitur

- Hero section dengan foto profil dan tombol booking WhatsApp
- Daftar menu potongan rambut pria (6 gaya)
- Galeri hasil potongan (8 foto dari Unsplash)
- Navigasi responsif mobile-friendly
- Dark theme dengan aksen gold (#c9a96e)

## Memulai

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Hasil build statis tersedia di folder `out/`.
